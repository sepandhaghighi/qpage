Overview

  • We developed QPage on Windows but it also run in Linux and Mac-OSX. QPage is written in Python3, in order to run it, you will need a python interpereter.

Dependencies

  • Download and install last version of python3 (*or use exe-file and skip this step*)

Installation

  • Directly
    • Download Qpage source or exe-file
    • Just Unzip It !
  • pip (Tutorial)
    • open cmd (in windows) or terminal (in linux) and type (python get-pip.py) to install pip
    • type (python pip install qpage)

Usage

  • Run setup (for the first use to create folders)
  • Put your profile picture in image folder (jpg , bmp , tiff , gif , png)
  • Put each page details in a text file with page name in doc folder (* In this step you can use [M] , [S] or [L] at the start and [center] at the end of each line to control font size and alignment )
  • Put your resume file (pdf format) in doc folder
  • Put your font file in font folder (*Optional) suggested site for --> Google Fonts
  • Put your icon file in image folder (*Optional) suggested site for --> IconArchive
  • Run again setup
  • Enter your name
  • Choose your profile image size
  • Choose your text and background color
  • You can preview your webpage
  • Finished!!! Now you can go and upload output folder contains directly in your host
