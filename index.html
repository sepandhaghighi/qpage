QPage Home Page

version 1.5.7

 

Free Project For Creating Academic Home page

 

Please Follow This Steps

 

(1) Download And Install Last Version Of Python ( * or use exe folder and skip this step * ) (1)

(2) Download QPage (2)

(3) Put your profile picture in image folder (3)

(4) Put your each page detail in a text file with page name in doc folder (* In this step you can use [M] , [S] or [L] at the start and [center] at the end of each line to control font size and alignment) (4)

(5) Put your resume file (pdf format) in doc folder (5)

(6) Put your font file in font folder (*Optional) (6)

(7) Run setup.py (7)

(8) Choose your profile image size (8)

(9) Choose your text and background color (9)

(10) Finished!!! Now you can go and upload output folder contains directly in your host (10)
